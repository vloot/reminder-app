## RemindMe - an app that will let you setup notification reminders for important tasks.

Features:

- Creating custom reminders
- Filtering reminders by day
- Modern, minimalist UI (inspired by [iRemind](https://dribbble.com/shots/20765248-iRemind-Reminder-Mobile-App))
- Localization support
- Regional settings customization
